# 🍦 Berry & Cream Ice Cream
A simple and responsive ice cream website built using HTML, CSS, and JavaScript. It features a clean design, smooth layout, and attractive product display.

## ✨ Features
- Responsive design for all devices  
- Modern UI with smooth scrolling  
- Ice cream product showcase  
- Clean and simple layout  

## 🛠️ Built With
- HTML  
- CSS  
- JavaScript  

## 🚀 How to Run
Just open `index.html` in your browser.
